    是秃子总会发光 Shiro

    shiro是一个不依赖Spring的安全框架，更加灵活，可用于认证、授权、密码加密、会话管理等

    # 认证

    # 认证原理

    参考SpringSecurity

    # shiro核心组件

    组件 功能
    Subject 用户
    SecurityManager 控制所有具体的交互，管理Subject，进行认证、授权、会话和缓存管理,核心模块，实际上活都是我干
    Authenticator 负责主体认证，可自定义实现认证策略
    Authorizer 决定主体是否有权限进行相应的操作，控制用户能访问应用中的哪些功能
    Realm 安全实体数据源(用户的信息、权限等)，用于获取安全实体的信息，可以是JDBC、LDAP、内存等实现
    SessionManager 管理主体与应用之间交互的数据，抽象了一个自己的Session来管理
    SessionDAO 用于会话的CRUD操作，可以实现自定义的SessionDAO，如JDBC写到数据库或放到Memcached中
    CacheManager 管理用户、角色、权限等的缓存，提高访问性能
    Cryptography 提供常见的加密组件，用于密码加密/解密

    # 授权

    # 原理

    1. Subject，委托给SecurityManager,SecurityManager接着委托给Authorizer;
2. Authorizer是真正的授权者，首先会通过PermissionResolver把字符串转换成相应的Permission实例;
3. 在进行授权之前，其会调用相应的Realm获取Subject相应的角色/权限用于匹配
4. Authorizer会判断Realm的角色/权限是否和传入的匹配.返回true/false

    public class Main {
    public static void main(String[] args) throws Exception {
        //创建和初始化SecurityManager
        IniSecurityManagerFactory factory = new IniSecurityManagerFactory("classpath:user.ini");
        SecurityManager securityManager = factory.getInstance();
        SecurityUtils.setSecurityManager(securityManager);

        //获取subject对象
        Subject subject = SecurityUtils.getSubject();

        //创建token对象
        AuthenticationToken authenticationToken = new UsernamePasswordToken("java", "123");

        //完成登录
        try {
            subject.login(authenticationToken);
            System.out.println("登录成功");
            //判断角色权限
            if (subject.hasRole("vip")) {
                System.out.println("vip资源，认证且拥有vip权限可访问");
            } else {
                System.out.println("没有vip权限");
            }
            //判断角色权限
            if (subject.hasRole("admin")) {
                System.out.println("管理员资源");
            } else {
                System.out.println("没有管理员权限");
            }
        } catch (Exception e) {
            System.out.println("登录失败");
        }
    }
}

    # 自定义登录

    • 前面的认证demo使用的是默认实现
    • 根据前面认证流程修改
    public class MyRealm extends AuthorizingRealm {

    @Override
    protected AuthenticationInfo doGetAuthenticationInfo(AuthenticationToken token) throws AuthenticationException {
        //token获取用户信息
        String username = token.getPrincipal().toString();
        String password = token.getCredentials().toString();
        //获取数据库中的用户信息[名字、密码、权限]，这里模拟,密码自行加密
        String dbUsername = "java";
        String dbPassword = new Md5Hash("123", "shiro", 3).toHex();
      
        if (username.equals(dbUsername)) {
            //返回Info,参考SpringSecurity
            AuthenticationInfo info = new SimpleAuthenticationInfo(
                    token.getPrincipal(),
                    dbUsername,
                    ByteSource.Util.bytes("shiro"),
                    token.getPrincipal().toString());
            return info;
        }
        return null;
    }

    @Override
    protected AuthorizationInfo doGetAuthorizationInfo(PrincipalCollection principals) {
        return null;
    }
}
